# Roadmap: FMD2 Mobile

Planned future expansions and feature updates for FMD2 Mobile.

## Short-Term Goals
- Add support for multiple alternative Jsoup sources (MangaDex, MangaFox, AsuraScans).
- Implement a dynamic plugin loader that loads scraper implementations from external DEX/APK files.
- Enable automatic chapter checks on startup for favorited manga.

## Medium-Term Goals
- Create custom category tags and custom sorting orders in the library.
- Implement an automated database backup and restore manager.
- Optimize image compression options to fit more files in local storage.

## Long-Term Goals
- Integrate a full-screen desktop mode.
- Establish cross-platform sync capabilities using self-hosted services.
